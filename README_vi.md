# Kế hoạch học An ninh mạng trong 90 ngày

## Giới thiệu

Chào mừng bạn đến với thử thách **90 Ngày Học An ninh mạng (Cybersecurity)**!

Kho mã nguồn này cung cấp một lộ trình học tập có cấu trúc trong 90 ngày, được thiết kế để giúp người học xây dựng nền tảng vững chắc về an ninh mạng.

## Mục tiêu

- Xây dựng nền tảng vững chắc về an ninh mạng.
- Tích lũy kinh nghiệm thực hành.
- Chuẩn bị cho các chứng chỉ Network+ và Security+.
- Tìm hiểu Linux, Python, Cloud Security và Ethical Hacking.

## Lộ trình

- Ngày 1–7: Network+
- Ngày 8–14: Security+
- Ngày 15–28: Linux
- Ngày 29–42: Python
- Ngày 43–56: Phân tích lưu lượng mạng
- Ngày 57–63: Git
- Ngày 64–70: ELK Stack
- Ngày 71–77: Cloud Platforms
- Ngày 85–90: Ethical Hacking
- Ngày 91–92: CV 1 trang
- Ngày 93–95: Tìm việc và nộp hồ sơ
